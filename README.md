# node_blog
#####基于 Node.JS 的博客应用开发


### 一、程序中使用到的：
> * Node.JS      _(v0.10.33)_
> * Express      _(v4.12.1)_
> * MongoDB      _(v3.0.1)_
> * jQuery       _(v1.11.3)_
> * Bootstrap    _(v3.3.5)_

### 二、如何运行程序：
#####1. 首先需要启动 *MongoDB* 服务:

```bash
$ mongod --config /usr/local/etc/mongod.conf
```

#####2. 然后进入程序目录并运行应用：

```bash
$ cd node_blog && ./bin/www
```
