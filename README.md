> * Express&ensp;&ensp;&ensp;&ensp;&ensp;[v4.12.1]

### 二、如何运行程序：
#####1. 首先需要启动 *MongoDB* 服务:

```bash
$ mongod --config /usr/local/etc/mongod.conf
```

#####2. 然后进入程序目录并运行应用：

```bash
$ cd node_blog && ./bin/www
```
