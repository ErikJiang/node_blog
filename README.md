# node_blog

[beta 实验版本]

运用 Node.JS、Express、MongoDB、Bootstrap 进行博客应用开发


### 如何使用

1. `git clone https://github.com/JiangInk/node_blog.git`

2. `cd node_blog/`

3. `npm install && bower install`

### 如何运行
开启一个终端，运行 _MongoDB_ 服务：

	$ mongod --config /usr/local/etc/mongod.conf

使用 `Gulp` 工具构建并运行应用服务:

    $ gulp
